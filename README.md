# 📋 Overview

**Face The Facts** is an AI-powered employee wellbeing monitoring system designed to detect **fatigue, stress, drowsiness, distraction, and emotional states** using computer vision and deep learning. The system continuously analyzes live webcam feeds to monitor employee wellness in real time while providing actionable insights through an interactive dashboard.

The application helps organizations improve employee productivity, workplace safety, and mental wellbeing by identifying early signs of fatigue and stress without requiring wearable devices.

## 🚀 Usage

### Installation

```bash
git clone https://github.com/yourusername/Face_The_Facts.git

cd Face_The_Facts
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run the Application

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
